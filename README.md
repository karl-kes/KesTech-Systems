# KesTech Systems

A small thank-you page for KesTech Systems, now closed.

Live at [kestechsystems.ca](https://kestechsystems.ca/).

## Structure

```
.
├── index.html
├── style.css
└── logos/
    ├── favicon.ico
    ├── favicon-16x16.png
    ├── favicon-32x32.png
    └── apple-touch-icon.png
```

Drop your existing favicon files into `logos/` and the site is complete.
